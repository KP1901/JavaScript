✅ 1) extends – Links Child Prototype to Parent Prototype

✅ Corrected:
The extends keyword sets up inheritance by linking Child.prototype to Parent.prototype, so that the child class inherits all methods from the parent class.

✅ 2) constructor() – Initializes Instance Properties

✅ Corrected:
-The constructor() is a special method used to initialize instance properties when a   class is instantiated. It acts like a function constructor in older JavaScript.
-✔️ constructor() — Initializes Properties for the Current Class

✅ 3) super() – Calls Parent Class Constructor

✅ Corrected:
In a subclass, super() is used inside the constructor to call the parent class's constructor. This allows the child to inherit and initialize properties from the parent.
