Browser Object Model (BOM) Examples

1. Window Object

The window object represents the browser window and is the global object in browsers.

2. Navigator Object

The navigator object contains information about the browser and system.

3. Location Object

The location object represents the current page URL and can be used for redirection.

4. History Object

The history object allows navigation through the browser's history.

5. Screen Object

The screen object provides information about the user's screen/monitor.

6. Dialog Boxes

BOM provides simple dialog boxes: alert, confirm, and prompt.
